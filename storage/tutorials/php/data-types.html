Data Types

Data types define what information is stored

The most used of them are:

Type Description Example
int Whole number. $var = 5;
double Floating point number. $var = 2.45;
String Sequence of characters (text). $var = 'PHP is awesome!';
char Represents a single symbol. $var = '*';
boolean Can have only 2 values - true or false. $var = true;
