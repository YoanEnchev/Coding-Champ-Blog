Data Types

Data types define what information is stored

The most used of them are:

Type Description Example
int Whole number. let num = 5;
double Floating point number. let num = 2.45;
String Sequence of characters (text). let text = 'JS is awesome!';
char Represents a single symbol. let symbol = '*';
boolean Can have only 2 values - true or false. let isAwesome = true;
