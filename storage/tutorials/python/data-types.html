Data Types

Data types define what information is stored

The most used of them are:

Type Description Example
int Whole number. num = 5;
double Floating point number. num = 2.45;
String Sequence of characters (text). text = 'Python is awesome!';
char Represents a single symbol. symbol = '*';
boolean Can have only 2 values - True or False. isAwesome = True;
