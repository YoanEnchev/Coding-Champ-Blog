For loop sets initial value for variable, condition and modification in one line.

Syntax

varName is initalized with value of start.

The loop repeats until varName reaches value of end .

Each iteration adds step value to varName.

step is optional parameter. If not specified the step of the loop is 1.

Examples

Let's have a program that prints each number from 1 to 10.
